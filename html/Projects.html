Another project I was able to be involved was an attempt to motorize mirror
mounts on a budget. Some may know that kinematic mirror mounts can be expensive.
Using 2 servo motors, an Arduino programmed with C++, and a 3D printed housing
a budget mirror mount now became available. With a program written to tilt the
X & Y Axis, percise angles were able to be read / achieved thanks to the program,
This kind of information can be especially important when handling lasers, or photonics.
